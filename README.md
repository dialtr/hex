# hextool

## About

**hextool** is a console application that performs analysis and various
operations on files that are in the Intel HEX (IHEX) format. It can
understand all official formats (I8HEX, I16HEX, and I32HEX), perform
merge operations while reporting in detail on conflicts, and generate
binaries from one or more files.
